Welcome to our WebApi Project. 

Please make sure to cd into our project with:

cd ./WebApi

In order to run our program please build first with: 

npm run build

Then start a server with 

cd compiled

node index.js

All Testing for our specifications can be found in the testAPI.http file.

A quick note about our project. In the compiled section all js files are
the most up to date compiliation of our project as JS, in case our TS files
do not compile properly once submitted.
